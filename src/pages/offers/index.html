Appels d'Offres Oriupe — le processus
01
Client soumet son projet
Le client décrit son besoin, budget et délai. Notre équipe examine et valide le projet sous 24-72h ouvrées.
02
Freelances postulent
Les freelances qualifiés soumettent leurs offres avec budget, portfolio et délai proposé. Chaque offre est horodatée.
03
Client choisit son freelance
Le client compare les offres, consulte les portfolios et sélectionne le freelance idéal. Échange de messages sécurisé.
04
Paiement Escrow & livraison
Le budget est bloqué en Escrow Oriupe. Libéré uniquement à la validation du livrable. Zéro risque pour les deux parties.

Description du projet

Nous recherchons une équipe de développement pour concevoir et développer une plateforme e-commerce B2B complète dédiée à la distribution agricole en Afrique de l'Ouest.

La plateforme devra permettre aux distributeurs de passer des commandes en ligne, suivre leurs livraisons en temps réel, gérer leur crédit client, et effectuer des paiements via Orange Money, MTN MoMo et Wave. Une interface admin complète pour la gestion des catalogues, stocks et rapports est également requise.

Compétences requises
React.js / Next.js Node.js / Express PostgreSQL Mobile Money API CinetPay / Wave Docker AWS / Vercel
